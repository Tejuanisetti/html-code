Example of HTML Table with border, rowspan, and colspan

Sr. No Name Marks
1 John Doe Math: 90 Science: 85
2 Jane Smith Math: 88 Science: 92
3 Math: 75 Science: 80

Explanation:

  • <table border="2">: Creates a table with border thickness 2.
  • <tr>: Defines a table row.
  • <th>: Defines a table header cell (bold and centered).
  • <td>: Defines a table data cell.
  • colspan="2": Merges two columns into one cell.
  • rowspan="2": Merges two rows into one cell vertically.
